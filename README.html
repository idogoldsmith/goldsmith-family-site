איך מעלים את האתר לאוויר בחינם

GitHub Pages

  • צור ריפו חדש ב־GitHub
  • העלה את כל קבצי התיקייה כמו שהם
  • הגדר Pages על branch ראשי
  • האתר יעלה בכתובת חינמית

Cloudflare Pages

  • פתח פרויקט חדש
  • גרור את התיקייה או חבר GitHub
  • הגדר root לתיקייה הראשית
  • Deploy ללא build

גיבוי

  • שמור את ה־ZIP המקורי
  • שמור את התיקייה הזו ב־Git
  • אפשר להעביר לכל אחסון סטטי אחר

העמוד הראשי הוא index.html. כל הנכסים הם יחסיים, כך שאפשר לפרוס את האתר כמו שהוא.

האתר נבנה מחדש מתוך הארכיון הישן של שלמה גולדשמיט, בעיצוב מודרני ובהיר.
מוכן לפריסה ל־GitHub Pages או Cloudflare Pages.
